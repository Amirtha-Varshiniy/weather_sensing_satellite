# Weather Sensing Mini Satellite

This project is a compact weather sensing and environmental monitoring system developed using Raspberry Pi Pico and MicroPython.

## About the Project
The system collects environmental and motion-related data using multiple sensors integrated with Raspberry Pi Pico. Sensor readings are processed and stored using an SD card module for future analysis.

The project was inspired by miniature satellite monitoring systems and IoT-based environmental sensing applications.

## Components Used
- Raspberry Pi Pico
- MPU Sensor
- BPU Sensor
- Smoke Sensor
- SD Card Module
- Jumper Wires
- Breadboard

## Technologies Used
- MicroPython
- Embedded Systems
- Sensor Interfacing
- Data Logging

## Features
- Real-time environmental monitoring
- Motion and orientation sensing
- Smoke detection
- Data storage using SD card
- Compact embedded system design

## Learning Outcome
This project improved my knowledge in embedded systems, MicroPython programming, sensor integration, and IoT fundamentals.
